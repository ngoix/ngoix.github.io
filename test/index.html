Bienvenue sur la page web du séminaire TesT !

Le séminaire TesT a lieu tous les mois à Telecom ParisTech (site Dareau) ou à l'université Paris 6. Il est l'occasion pour les doctorants ou post-doctorants de présenter leurs travaux de recherche. Les exposés sont précédés par un tutoriel lors duquel un doctorant présente les bases et les résultats classiques du domaine abordé.
